Description

Saya adalah seorang pengembang web front-end dengan minat yang kuat dalam menciptakan pengalaman pengguna yang menarik dan fungsional. Mahir dalam berbagai teknologi web seperti HTML, CSS, dan JavaScript, saya menikmati proses merancang serta mengembangkan website yang responsif dan mudah digunakan.

Saya memiliki pengalaman dalam bekerja dengan framework dan library seperti Bootstrap, Tailwind CSS, dan React JS. Selain itu, saya telah membangun berbagai proyek website, termasuk yang menggunakan LocalStorage dan API sebagai sumber data. Saya juga aktif mengikuti kursus di Dicoding, Gamelab, dan Neotic Id untuk terus mengembangkan keterampilan saya.

Dengan fokus pada kualitas dan pengalaman pengguna, saya berusaha menghadirkan solusi kreatif dan efektif dalam setiap proyek. Saya siap berkolaborasi dengan tim serta terbuka terhadap tantangan baru dalam dunia pengembangan web yang terus berkembang.
